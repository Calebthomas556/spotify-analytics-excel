# Spotify Analytics (Excel)

## Overview
This project analyzes personal Spotify listening data using Excel to identify trends in listening behavior, top artists, genres, and track popularity.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations
- Basic data cleaning

## Dataset
- File: `Spotify_Analytics.xlsx`
- Source: Spotify listening history
- Key fields: artist, track, genre, play count, duration, date

## Key Questions Explored
- Which artists and songs are listened to most frequently?
- How does listening behavior change over time?
- Are there patterns in listening duration or frequency?

## Key Insights
- A small group of artists accounts for a large share of total plays
- Top Songs differ by minutes played and total number of plays
- Listening activity is more dominant later in the day
